# two-five-one
http://twofive.one

Record and play back MIDI.

## Run Locally
1. Download the code.  
`$ git checkout https://github.com/MarkMendell/two-five-one.git`
2. Enter the project directory.  
`$ cd two-five-one`
3. Serve the site [locally on port 2511](http://localhost:2511).  
`$ python -m SimpleHTTPServer 2511`
